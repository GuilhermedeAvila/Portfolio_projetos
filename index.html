Sobre

    • Projetos em Ciência de Dados

    Rossman Sales

    Projeto com intuito de realizar a previsão de vendas das próximas 6 semanas da empresa do ramo farmacêutico Rossmann. Usando dados históricos das vendas para treinar os modelos de Machine Learning. Foi entregue o valor da previsão, estimado com cenários positivos e negativos. Utilizei 5 modelos diferentes de machine learning no final o modelo escolhido foi o XGBoost.

    As ferramentas utilizadas foram:

  • Python, Pandas, Numpy,Jupyter Notebook, VSCode e Pyenv.
  • BorutaPy, Scikit-Learn e Scipy.
  • Git e GitHub.
  • Matplotlib e Seaborn.

    • House Rocket

    Projeto se trata de uma simulação sobre ciência de dados em uma empresa do ramo imobiliário. Como resultado do projeto, resolvemos alguns problemas de gerenciamentos de informações, e conseguimos aumentar a margem de lucro da empresa em um total de 19.89%. Foi disponibilizado a análise da base de dados em um Dashboard online pelo StreamlitCloud. A recomendação de compra e venda dos imóveis foi disponibilizada em arquivos para excel.

    As ferramentas utilizadas foram:

  • Python, Pandas, Numpy, Seaborn, Matplotlib, Plotly, Streamlit.
  • VSCode, JupyterNotebook.
  • Git e GitHub.
  • Mapa interativo com Plotly e Folium.
  • StreamlitCloud.
